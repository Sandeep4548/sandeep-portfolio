## 🔬 Engineering Highlights

<!-- FIX: Added blank line after <br/> inside each <details> block — GitHub requires this for markdown to parse inside collapsible sections -->

<details>
<summary><strong>🗂️ Dynamic PLP Tab System — Advanced Frontend Pattern</strong></summary>

<br/>

Built a zero-page-reload PLP tab system with these features:

- Active tab automatically moves to **position 1**
- Grid filters update dynamically
- **URL slug updates** without browser reload via `?tab=` param
- Full state persists on page refresh

This is senior-level frontend architecture — not just a filter button.

</details>

<details>
<summary><strong>💀 Adaptive Skeleton Loader — CLS Elimination</strong></summary>

<br/>

Solved Cumulative Layout Shift (CLS) with an intelligent skeleton loader that:

- **Reads collection metafield** to detect grid layout (3-col or 4-col)
- Mirrors the exact real grid structure before products load
- Completely eliminates layout jumps — measurably better Core Web Vitals score

</details>

<details>
<summary><strong>📜 Ajax Scroll Pagination — URL-Persistent State</strong></summary>

<br/>

Built infinite-scroll pagination that does not break UX:

- Initial load: **8 products**
- Subsequent loads via Ajax with GIF loader
- **`?page=N`** URL state updates silently on scroll
- Page state persists after refresh — users never lose their scroll position

</details>

<details>
<summary><strong>🛒 Metafield-Driven Architecture — No-Code PDP/PLP</strong></summary>

<br/>

Replaced all hardcoded product page logic with **dynamic metafield and metaobject systems**:

- Merchants configure bundles, tabs, pricing rules, and content via Shopify Admin
- **Zero developer intervention** needed for content updates
- Scales across multi-brand, multi-region storefronts

</details>

<details>
<summary><strong>🌍 Multi-Language Store — EN-US and FR-CA</strong></summary>

<br/>

Implemented full multilingual storefront support using:

- **Translate and Adapt** for content localization
- **Storefront API** for locale-aware data fetching
- Regional promotional pricing for multi-region campaigns

</details>
